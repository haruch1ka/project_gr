# Gemini モデル移行記録

## 経緯

`gemini-2.0-flash` が 2026年6月1日をもって廃止され、API が 503 エラーを返すようになった。

## 廃止モデル

| モデルID | 廃止日 |
|----------|--------|
| `gemini-2.0-flash-001` | 2026-06-01 |
| `gemini-2.0-flash-lite-001` | 2026-06-01 |

## 採用モデル

**`gemini-2.5-flash`**

選定理由：
- 無料枠で利用可能
- 低レイテンシ・価格性能比重視（大量タスク向け）
- 本プロジェクトの用途（JSON生成・知識分類・確信度更新）に適合

## 無料枠で使えるFlashモデル一覧（2026年6月時点）

| モデルID | 特徴 |
|----------|------|
| `gemini-3.5-flash` | 最新・最高性能 |
| `gemini-3.1-flash-lite` | 軽量・高速 |
| `gemini-2.5-flash` | 安定版・低レイテンシ ← **採用** |
| `gemini-2.5-flash-lite` | 超軽量 |

## 変更箇所

- `back/src/router/gemini.ts` の `MODEL` 定数を `gemini-2.5-flash` に変更
- `CLAUDE.md` の技術スタック表を更新
